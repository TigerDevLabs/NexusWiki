# Módulos

NexusSlime está organizado en **módulos de funcionalidades**, cada uno gestionando un sistema de juego distinto. Todos los módulos están incluidos en el JAR final y habilitados por defecto.

---

| Módulo | Descripción |
| --- | --- |
| [Core](core.md) | Ítems personalizados, sistema PDC, registro de ítems, mochilas, waypoints |
| [Esenciales](essentials.md) | Homes, warps, TPA, AFK, cárcel, +40 comandos QoL |
| [Economía](economy.md) | Dinero, créditos, /sell, /worth, /baltop |
| [Clanes](clans.md) | Creación de clan, territorio, mejoras, cofre de clan, chat de clan |
| [Seguridad](security.md) | Auth BCrypt, anti-bot CAPTCHA, detección de mods, anti-lag, anti-dupe |
| [Discord](discord.md) | Bot JDA, vinculación de cuenta, sincronización de roles, avisos en vivo |
| [Crystal Defense](crystal-defense.md) | Minijuego cooperativo de defensa de Cristales Ender por oleadas |
| [Votifier](votifier.md) | Servidor Votifier V1/V2 independiente, rachas de votos, clasificación |
| [Sueños](dreams.md) | Sistema de cutscene de sueño con variantes de sueño y pesadilla |
| [Protecciones](protections.md) | Reclamación de regiones estilo WorldGuard, 28 banderas de protección, duelo |
| [Mobs Personalizados](custom-mobs.md) | Jefes definidos en YAML con tablas de botín, formas de IA y huevos de spawn |
| [Crafteo Infinito](infinity-crafting.md) | Estaciones de crafteo multibloques con formato de receta YAML |
| [Integración Twitch](twitch.md) | Vinculación de cuenta, alertas en vivo, relay de chat, sorteos para espectadores |
| [TAB](tab.md) | Encabezado/pie TAB, prefijo de rango, ping, marcador, MOTD |
| [Chat](chat.md) | Sistema de 4 canales (local, global, staff, report) con prefijo de LuckPerms |
| [Hologramas](holograms.md) | Displays flotantes de texto, ítems y cabezas de jugador via YAML |
| [Traits](traits.md) | Sistema de Cartas del Tarot — 3 cartas por jugador, aleatorio ponderado, reroleable |
| [RNG](rng.md) | Giro diario, bloques de suerte, gacha via Pergaminos de Investigación, eventos de servidor |
| [Cofres](crates.md) | Cofres de botín con llaves, animaciones de apertura y tablas de recompensas por cofre |
| [Encantamientos](enchantments.md) | 175 encantamientos personalizados, 6 rarezas, 10 tipos de disparador, definidos en YAML |
| [Almacenamiento ME](ae.md) | Sistema de almacenamiento en red estilo Applied Energistics |
| [Energía](energy.md) | Red de energía en Julios — generadores, capacitores, cables e integración con máquinas |
| [Estructuras](structures.md) | Inyección de botín personalizado en 11 tipos de estructuras vanilla |
| [Web & RGPD](web.md) | Integración con tienda virtual, entrega de kits VIP, webhooks de pago, cumplimiento RGPD |
